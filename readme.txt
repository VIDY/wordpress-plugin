With just a hold, users can now reveal tiny hyper-relevant videos hidden behind the text of any page on the web, unlocking a whole new dimension to the internet.

== Description ==

All you have to do, to insert Vidy into your website is:

*	Copy your Application ID from:
	[Vidy Dashboard](https://dashboard.vidy.com/settings)
*	Paste the Application ID into Settings > Vidy > Application ID
*	Save the settings and the plugin inserts the required code to setup Vidy on your website.

== Installation ==
= Automatic Installation =
*	Install the plugin through the WordPress plugins screen
*	Use the Settings->Vidy screen to paste your Application ID

= Manual Installation =
*	Upload the plugin files to the `/wp-content/plugins/` directory
*	Activate the plugin through the 'Plugins' screen in WordPress
